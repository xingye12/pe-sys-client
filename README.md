# 中小学体育考试系统

基于 Vue 3 + TypeScript + Element Plus 开发的体育考试管理系统

## 功能特性

### 管理员端
- 📊 数据大屏：智慧看板展示整体数据
- 🏫 班级管理：管理所有班级和学生信息
- 📹 视频管理：批量上传和AI智能分析
- 📝 任务发布：为各班级发布考试任务

### 教师端
- 📈 班级总览：关键指标和趋势分析
- 👨‍🎓 学生管理：成绩查询、详细分析
- 📊 数据导出：导出班级和学生数据

### 学生端
- 🎯 我的成绩：查看个人考试成绩
- 📉 历史追踪：成绩趋势和进步分析
- 📺 学习资源：标准动作视频教学

## 技术栈

- **前端框架**: Vue 3 (Composition API)
- **UI组件库**: Element Plus
- **图表库**: ECharts
- **状态管理**: Vue Router
- **HTTP客户端**: Axios
- **构建工具**: Vite
- **语言**: TypeScript

## 快速开始

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```

### 生产构建
```bash
npm run build
```

### 预览构建
```bash
npm run preview
```
